# Mode: compare

Use this when the user wants to compare multiple roles or previously evaluated opportunities.

## Weighted matrix

| Dimension | Weight | 5 means | 1 means |
|-----------|--------|---------|---------|
| North Star alignment | 25% | Exact target role | Not aligned |
| CV match | 15% | 90%+ fit | Under 40% fit |
| Level | 15% | Staff+ | Junior |
| Compensation | 10% | Top quartile | Below market |
| Growth path | 10% | Clear next-step upside | Dead end |
| Remote quality | 5% | Strong remote setup | Onsite only |
| Employer quality | 5% | Strong signal | Red flags |
| Tech stack quality | 5% | Modern, relevant stack | Legacy |
| Time to offer | 5% | Fast process | Very slow |
| Cultural fit | 5% | Builder-friendly | Bureaucratic |

For each role:
1. Score each dimension.
2. Compute the weighted total.
3. Rank the options.
4. Give a recommendation that balances quality and time-to-offer.

If the roles are not already in context, ask for the text, URLs, or report references first.
