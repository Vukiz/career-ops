# Mode: tracker

Read and display `data/applications.md`.

## Tracker format

```markdown
| # | Date | Company | Role | Score | Status | PDF | Report |
```

Canonical statuses:
- `Evaluated`
- `Applied`
- `Responded`
- `Interview`
- `Offer`
- `Rejected`
- `Discarded`
- `SKIP`

If the user asks to update a status, edit the relevant row or use the dashboard flow.

Also show:
- total applications,
- counts by status,
- average score,
- PDF coverage,
- report coverage.
